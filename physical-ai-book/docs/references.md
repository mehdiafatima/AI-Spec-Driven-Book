# References

This section compiles a comprehensive list of official documentation, peer-reviewed articles, and tutorials relevant to Physical AI, Humanoid Robotics, ROS 2, Digital Twins, NVIDIA Isaac Sim, and related technologies. These resources are essential for deeper understanding and continued learning.

## Official Documentation

### ROS 2

-   **ROS 2 Documentation**: The primary resource for all things ROS 2, including installation, tutorials, and API references.
    -   [docs.ros.org](https://docs.ros.org/)
-   **ROS 2 Galactic Geochelone (LTS)**: Specific documentation for the Galactic long-term support release.
    -   [docs.ros.org/en/galactic/](https://docs.ros.org/en/galactic/)
-   **ROS 2 Humble Hawksbill (LTS)**: Specific documentation for the Humble long-term support release.
    -   [docs.ros.org/en/humble/](https://docs.ros.org/en/humble/)

### NVIDIA Isaac Sim & Omniverse

-   **NVIDIA Isaac Sim Documentation**: Comprehensive guide for using Isaac Sim, including Python APIs, USD, and ROS 2 integration.
    -   [docs.omniverse.nvidia.com/isaacsim/latest/](https://docs.omniverse.nvidia.com/isaacsim/latest/)
-   **NVIDIA Omniverse Documentation**: General documentation for the Omniverse platform, USD, and its various applications.
    -   [docs.omniverse.nvidia.com/](https://docs.omniverse.nvidia.com/)
-   **Universal Scene Description (USD) Documentation**: Official documentation for Pixar's USD format.
    -   [graphics.pixar.com/usd/docs/index.html](https://graphics.pixar.com/usd/docs/index.html)

### Gazebo Simulation

-   **Gazebo Documentation**: Official documentation for the Gazebo simulator, including tutorials, models, and plugins.
    -   [classic.gazebosim.org/](http://classic.gazebosim.org/) (for Gazebo Classic)
    -   [gazebosim.org/docs](https://gazebosim.org/docs) (for Gazebo Ignition/Fortress and newer versions)
-   **`ros_gz` Bridge Documentation**: For integrating Gazebo with ROS 2.
    -   [github.com/ros-simulation/ros_gz](https://github.com/ros-simulation/ros_gz)

### Unity Technologies

-   **Unity Manual**: Official manual for the Unity game engine.
    -   [docs.unity3d.com/Manual/index.html](https://docs.unity3d.com/Manual/index.html)
-   **Unity Robotics Hub**: Resources for using Unity in robotics, including ROS-Unity integration.
    -   [github.com/Unity-Technologies/Unity-Robotics-Hub](https://github.com/Unity-Technologies/Unity-Robotics-Hub)

## Peer-Reviewed Articles and Books

-   **"Probabilistic Robotics" by Sebastian Thrun, Wolfram Burgard, Dieter Fox**: A foundational textbook on mobile robot perception and control.
-   **"Planning Algorithms" by Steven M. LaValle**: Comprehensive text on motion planning.
-   **"ROS Robotics Projects" (various editions)**: Practical guides for building projects with ROS.
-   **Research Papers on VLA (Vision-Language-Action) Models**:
    -   Search on Google Scholar or arXiv for recent publications from institutions like Google DeepMind, OpenAI, Stanford, MIT, and Carnegie Mellon. Keywords: "Vision-Language-Action," "Embodied AI," "Robot Learning from Language," "Humanoid Control."

## Tutorials & Guides

-   **The Construct ROS 2 Tutorials**: Extensive video and text tutorials for learning ROS 2.
    -   [theconstructsim.com/ros-2-academy/](https://www.ros2.academy/)
-   **NVIDIA Jetson Documentation**: Guides for setting up and developing on NVIDIA Jetson edge devices.
    -   [developer.nvidia.com/embedded/jetson-developer-kits](https://developer.nvidia.com/embedded/jetson-developer-kits)
-   **PyTorch/TensorFlow Documentation**: For deep learning frameworks used in AI models.
    -   [pytorch.org/docs/](https://pytorch.org/docs/)
    -   [tensorflow.org/learn](https://www.tensorflow.org/learn)
